# 🔗 ربط قاعدة البيانات

## الخطوات:

### 1️⃣ الحصول على قاعدة بيانات مجانية

يمكنك استخدام أحد هذه الخدمات المجانية:

**خيار 1: Supabase** (موصى به ⭐)
- اذهب إلى: https://supabase.com
- أنشئ حساب مجاني
- أنشئ مشروع جديد (New Project)
- اختر اسم المشروع وكلمة المرور
- انتظر حتى يتم إنشاء قاعدة البيانات (دقيقتين تقريباً)
- اذهب إلى: **Settings** → **Database**
- انسخ **Connection String** من قسم "Connection string" - اختر "URI"

**خيار 2: Railway**
- اذهب إلى: https://railway.app
- أنشئ حساب مجاني
- New Project → Provision PostgreSQL
- اذهب إلى: **Variables** → **DATABASE_URL**
- انسخ القيمة

**خيار 3: Render**
- اذهب إلى: https://render.com
- أنشئ حساب مجاني
- New → PostgreSQL
- انتظر حتى يتم إنشاء قاعدة البيانات
- انسخ **External Database URL**

---

### 2️⃣ إضافة DATABASE_URL في ملف .env

افتح ملف `.env` في مجلد التطبيق وأضف:

```
DATABASE_URL=postgresql://user:password@host:5432/database
```

استبدل القيمة بـ Connection String الذي نسخته من الخدمة.

**مثال من Supabase:**
```
DATABASE_URL=postgresql://postgres.abcd1234:MyPassword123@aws-0-us-west-1.pooler.supabase.com:5432/postgres
```

---

### 3️⃣ تشغيل سكريبت إنشاء الجداول

بعد إضافة `DATABASE_URL`، افتح المتصفح واذهب إلى:

```
http://localhost:3000/api/init-db
```

أو في بيئة الإنتاج:
```
https://your-app-url.com/api/init-db
```

ستظهر رسالة **✅ تم إنشاء قاعدة البيانات بنجاح!**

---

### 4️⃣ تأكد من الربط

- افتح صفحة **الدردشة** → يجب أن تظهر رسائل تجريبية
- افتح صفحة **المشاريع** → يجب أن تظهر مشاريع تجريبية
- افتح صفحة **الوظائف** → يجب أن تظهر وظائف تجريبية

---

## ❌ حل المشاكل الشائعة

**المشكلة: خطأ "connection refused"**
- تأكد من أن `DATABASE_URL` صحيح
- تأكد من عدم وجود مسافات إضافية

**المشكلة: خطأ "password authentication failed"**
- تأكد من كلمة المرور الصحيحة في الـ URL
- إذا كانت كلمة المرور تحتوي على رموز خاصة (@, #, etc)، يجب ترميزها

**المشكلة: "DATABASE_URL غير موجود"**
- تأكد من حفظ ملف `.env`
- أعد تشغيل الخادم بعد تعديل `.env`

## 📊 الجداول التي تم إنشاؤها:

✅ **users** - المستخدمين  
✅ **chats** - الدردشات  
✅ **messages** - الرسائل  
✅ **projects** - المشاريع  
✅ **proposals** - العروض المقدمة على المشاريع  
✅ **jobs** - الوظائف  
✅ **job_applications** - طلبات التوظيف  
✅ **offers** - عروض الأسعار

## 🎯 بعد الربط

جميع صفحات التطبيق ستستخدم قاعدة البيانات الفعلية:
- ✅ تسجيل الدخول والتسجيل
- ✅ الدردشة والرسائل
- ✅ المشاريع والعروض
- ✅ الوظائف والتقديمات
- ✅ عروض الأسعار

---

**ملاحظة:** لا تشارك `DATABASE_URL` مع أحد - هي مثل كلمة السر لقاعدة البيانات!
